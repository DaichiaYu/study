## 使用原則

目前多數檔案是逐字稿或原始文字資料，建議保留原文作為 source archive，再依主題整理成 AI digest、教學模板或可執行方案。

建議後續每份資料維持三層：

1. **原始檔**：保留逐字稿與來源脈絡。
2. **AI digest**：整理成可被 AI 穩定理解與重複調用的知識結構。
3. **應用模板**：轉成可直接用於孩子學習、家長引導或系統設計的操作格式。
