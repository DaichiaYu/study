## 主要內容

### 學習方法

整理具體學習流程與學科方法，包含清單、計畫、時間管理、預習、聽課、作業、復習、改錯、刷題與語文／數學／英語策略。

### 自驅力

整理孩子如何從被動、防禦、抵觸，逐步回到有掌控感、有動力、有慣性、能重新啟動的狀態。核心架構包含壓力系統、動力系統、慣性系統、喚醒系統。

### 德育課

整理德育、規則感、群體生活、邊界、紀律、非暴力與作業課互動相關資料。

## 使用原則

目前多數檔案是逐字稿或原始文字資料，建議保留原文作為 source archive，再依主題整理成 AI digest、教學模板或可執行方案。

建議後續每份資料維持三層：

1. **原始檔**：保留逐字稿與來源脈絡。
2. **AI digest**：整理成可被 AI 穩定理解與重複調用的知識結構。
3. **應用模板**：轉成可直接用於孩子學習、家長引導或系統設計的操作格式。
