## 主要內容

### 00_system：整理規則與格式

存放本倉庫的整理規則、標籤系統、AI digest 格式、案例卡格式與 NTMY story seed 格式。

### 家衛：學習與孩子操作系統

偏向孩子學習、自驅力、德育與具體操作方法。

- **學習方法**：清單、計畫、時間管理、預習、聽課、作業、復習、改錯、刷題與語文／數學／英語策略。
- **自驅力**：壓力系統、動力系統、慣性系統、喚醒系統。
- **德育課**：德育、規則感、群體生活、邊界、紀律、非暴力與作業課互動。
- **直播**：課後問答、學員案例、家庭／學校真實場景補充。

### 沈奕斐：父母鬆弛與家庭系統

偏向父母認知鬆綁、家庭教育、親密關係、性教育、社會學視角與愛的能力。

- **鬆弛父母**：密集母職、父母職責、內驅力、自控力、情緒力、青春期、手機、家庭關係等。
- **家庭教育**：規矩、表揚、情商、主動學習、興趣班、老人帶娃、二胎、家校矛盾等。
- **社會學愛情思維**：愛情腳本、性別、價值差異、長期關係、金錢勞動、衝突修復等。
- **性教育課**：成人性認知、性溝通、安全性、親密關係與孩子性教育。

## 使用原則

目前多數檔案是逐字稿或原始文字資料，建議保留原文作為 source archive，再依主題整理成 AI digest、教學模板或可執行方案。

建議後續每份資料維持三層：

1. **原始檔**：保留逐字稿與來源脈絡。
2. **AI digest**：整理成可被 AI 穩定理解與重複調用的知識結構。
3. **應用模板**：轉成可直接用於孩子學習、家長引導、家庭規則、親密關係或自我整理的操作格式。

## 整合使用方式

建議先用沈奕斐資料校準父母焦慮、密集母職、家庭系統與親密關係，再用家衛資料處理孩子具體學習、自驅力、德育與日常操作問題。

簡單說：

- **沈奕斐**：先判斷大人是不是太焦慮、太控制、太委屈、太用力。
- **家衛**：再判斷孩子具體卡在哪裡，以及怎麼拆解、激勵、建立流程。
- **00_system**：規定資料怎麼標籤、怎麼整理、怎麼轉成案例或 NTMY story seed。
