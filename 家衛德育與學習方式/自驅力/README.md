# 自驅力

這個資料夾整理「如何喚醒孩子的自驅力」相關資料，主軸是把孩子從被動、防禦、抵觸，逐步帶回有掌控感、有動力、有慣性、能重新啟動的狀態。

## 核心結構

原始課程可分為四大系統：

1. **壓力系統**：先處理毒性壓力與失控感，讓孩子從防禦狀態回到可溝通、可行動的狀態。
2. **動力系統**：在壓力系統穩定後，補上興趣、激勵、目標與前進感。
3. **慣性系統**：把好的行為變成流程、習慣與低耗能自動化。
4. **喚醒系統**：當孩子自驅力已經受損時，依照煩躁、疲憊、抵觸、放棄等程度選擇修復策略。

## 使用原則

這套內容有明確順序，不建議跳過壓力系統直接談動力或習慣。壓力系統是地基；孩子若長期處於失控、防禦、恐懼或抵觸狀態，後面的激勵、習慣、目標設計容易變成新的壓力源。

後續最適合的整理流程：

1. 保留原始母檔。
2. 依四大系統拆成小節原文。
3. 每一小節整理成 AI digest。
4. 再做一份「孩子狀態 → 對應系統 → 建議策略」的總索引。
