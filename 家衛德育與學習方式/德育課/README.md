# 德育課

這個資料夾整理德育、規則感、群體生活、邊界、紀律、非暴力與作業課互動相關資料。

這套內容和「自驅力」不同：

- **自驅力**偏向孩子能不能主動開始、持續投入、形成習慣。
- **德育課**偏向孩子如何在群體、規則、關係與邊界中穩定做人做事。

## 後續整理建議

這套德育課適合整理成三層：

1. **原文保留**：保留逐字稿，不修改。
2. **AI digest**：整理核心命題、規則、親子介入原則與容易誤解處。
3. **家庭應用版**：轉成家長可實際使用的說法與情境腳本。

優先整理順序建議：

1. 要不要聽話：釐清服從、判斷與自主的關係。
2. 底線與邊界：建立孩子在關係中的尊嚴與界線。
3. 紀律與習慣：建立群體生活中的規則感。
4. 是非觀：處理傷害、破壞規則、挫敗與撒謊等情境。
