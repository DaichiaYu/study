## 整理方向

目前資料以課程逐字稿與原始文字為主，後續建議依照以下方向加工：

1. **學習觀念**：例如清單、計畫、時間管理、閱讀、晨讀。
2. **學習流程**：例如預習、聽課、作業、復習、改錯。
3. **學科方法**：語文、數學、英語等不同學科的學習策略。
4. **自驅力與習慣**：如何讓孩子從被動完成任務，逐步轉向主動解決問題。
5. **德育與邊界**：包含聽話、發言權、合群、底線、禮儀、紀律、是非觀與邊界感。
6. **直播案例**：課後問答、學員案例、家庭／學校真實場景補充。

## 主要資料夾說明

### 學習方法

偏向具體學習流程與學科方法，包含清單、計畫、時間管理、預習、聽課、作業、復習、改錯、刷題與語文／數學／英語策略。

### 自驅力

偏向孩子能不能主動開始、持續投入、承受挑戰、形成習慣。核心架構分為壓力系統、動力系統、慣性系統、喚醒系統。

### 德育課

偏向孩子如何在群體、規則、關係與邊界中穩定做人做事，包含聽話、合群、底線、嚴格、禮儀、紀律、是非觀、作業課衝突與邊界感。

### 直播

偏向主課之外的課後問答與真實案例，適合在遇到具體問題時查找相近場景，例如作業做不完、注意力不集中、偷東西、青春期溝通、學校溝通等。
