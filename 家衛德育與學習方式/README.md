# 家衛德育與學習方式

這個資料夾存放家庭教育、德育、自學力、學習習慣與具體學習方法相關資料。

## 子目錄

- [學習方法](./學習方法/README.md)

## 整理方向

目前資料以課程逐字稿與原始文字為主，後續建議依照以下方向加工：

1. **學習觀念**：例如清單、計畫、時間管理、閱讀、晨讀。
2. **學習流程**：例如預習、聽課、作業、復習、改錯。
3. **學科方法**：語文、數學、英語等不同學科的學習策略。
4. **自驅力與習慣**：如何讓孩子從被動完成任務，逐步轉向主動解決問題。

## AI 使用建議

不要直接把整包逐字稿當長期主讀資料。比較適合的流程是：

- 原始逐字稿：保留證據與完整脈絡。
- AI digest：抽出核心命題、操作框架與可泛化模型。
- 家庭應用版：改寫成家長可以實際使用的引導方案。
