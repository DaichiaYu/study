## 使用方式

建議不要把直播當主線讀，而是配合主課查用：

1. 先從主課判斷問題屬於學習方法、自驅力、德育或家庭互動。
2. 再到直播資料中找對應案例。
3. 最後整理成「案例 → 問題類型 → 原因判斷 → 可用策略 → 注意事項」。

## 後續整理建議

這批直播資料適合整理成案例索引，例如：

- **作業與學習**：作業做不完、抄作業、過關訓練出錯、注意力不集中。
- **德育與行為**：偷東西、畏難情緒、錯了又錯、習慣不好。
- **親子與青春期**：青春期說話、成年人衝突後怎麼跟孩子說、媽媽不讓托管。
- **學校溝通**：老師針對我、特殊情況如何與學校溝通。
- **分層學習**：學霸、中等生、學渣不同需求與交流方式。
