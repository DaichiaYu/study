## 後續整理建議

每篇逐字稿可再整理成三種版本：

1. `source`：原始逐字稿，不修改。
2. `ai_digest`：給 AI 長期讀的核心命題、框架、操作原則。
3. `family_playbook`：轉成家長可實際引導孩子的做法。

目前最值得先整理的起點：

1. 清單與計畫：建立孩子學習安排的底層系統。
2. 作業三篇：把作業從被動任務轉成學習資源。
3. 復習與改錯：建立錯題、回顧、過關標準。
4. 語文／數學／英語：拆成學科專用方法。
