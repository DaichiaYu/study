# 沈奕斐家庭父母與夫妻

這個資料夾整理沈奕斐老師相關課程資料，主軸包含父母鬆弛、家庭教育、親密關係／愛情社會學，以及性教育。

這套資料和「家衛德育與學習方式」的定位不同：

- **家衛**偏向孩子學習、自驅力、德育與具體操作系統。
- **沈奕斐**偏向父母認知鬆綁、家庭系統、親密關係、社會學視角與愛的能力。

## 使用定位

### 01鬆弛父母

用來校準父母焦慮、密集母職、陪伴焦慮、學業焦慮、青春期焦慮與家庭關係中的內耗。適合當作「父母先鬆綁」的上層心法。

### 02家庭教育

偏向家庭教育的通用框架與日常問題處理，包含規矩、表揚、情商、興趣班、手機、老人帶娃、二胎、家校矛盾等。

### 03社會學愛情思維

偏向親密關係、婚姻、愛情腳本、價值差異、長期關係、金錢與勞動、父母干涉、衝突修復等議題。適合用於成人關係與家庭系統理解。

### 04性教育課

偏向成人性認知、親密關係中的性溝通、安全性、性教育、性危機與孩子性教育。適合做家庭性教育與親密關係界線的資料基礎。

## 與家衛資料的整合方式

建議使用順序：

1. 先用沈奕斐資料檢查父母是否過度焦慮、過度控制、過度自責，或把孩子當成家庭 KPI。
2. 再用家衛資料判斷孩子具體卡在壓力系統、動力系統、慣性系統、德育邊界，還是學習方法。
3. 最後才把問題轉成可執行的家庭流程、學習設計或對話腳本。
